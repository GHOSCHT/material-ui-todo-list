## Getting Started

### Installation

1.  Clone the repo

```sh
git clone https://github.com/your_username_/Project-Name.git
```

2.  Install NPM packages

```sh
npm install
```

## Development server

Start the development server

```sh
npm start
```

## Deployment

Deploy to GitHub Pages

```sh
npm deploy
```
